# Sistem Validasi Registrasi Mahasiswa (SOLID)

## Deskripsi
Project ini mengimplementasikan sistem validasi registrasi mahasiswa
menggunakan prinsip SOLID, Docstring, dan Logging.

## Cara Menjalankan
```bash
python registration_service.py
```

## Contoh Output
```
INFO - Memulai proses validasi registrasi mahasiswa.
INFO - Validasi SKS berhasil.
INFO - Validasi prasyarat berhasil.
INFO - Registrasi mahasiswa diterima.
```
